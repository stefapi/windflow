# EPIC-021 : Command Palette

**Statut :** TODO
**Priorité :** Basse

## Vision
Palette de commandes pour navigation rapide. Les utilisateurs peuvent ouvrir la palette avec un raccourci clavier, rechercher et naviguer vers n'importe quelle page ou exécuter des actions rapides.

## Liste des Stories liées
- [ ] STORY-001 : Ouvrir la palette de commandes avec un raccourci clavier
- [ ] STORY-002 : Rechercher et naviguer vers n'importe quelle page
- [ ] STORY-003 : Exécuter des actions rapides depuis la palette

## Notes de conception
- Raccourci clavier : Ctrl+K ou Cmd+K
- Recherche floue sur les pages et actions
- Navigation au clavier (flèches, Enter, Escape)
- Actions rapides : créer conteneur, ouvrir terminal, etc.
- Persistance des commandes récentes
- Support des raccourcis personnalisés
- Intégration avec le système de permissions (masquer les actions non autorisées)
